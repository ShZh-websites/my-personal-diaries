通过高考我进入了武汉大学读书，我也是通过自己的分数才知道这个大学。刚进入大学的我对校园生活很憧憬，大一这一年我所做的事情基本可以概括为两类：第一是享受美好的校园生活，第二是打计算机的各种基础。



# 军训

军训是在武大的第一个大事件。其实之前高中也办过军训，但是和大学里的显然没法比。大学军训的持续时间非常长，每一天都被排满了。

军训的第一天，我们被拉去梅操观看开幕仪式。按照规定，期间是不能喝水的，我们的教官干脆禁止我们带水。于是在有“火炉”之称的武汉，我们滴水不沾地过了一上午。回到工学部后，团长根据之前在开幕仪式提到的要求，对我们进行了训练。因为训练的场地并不是我们一开始集合的地方，所以还是喝不了水。团长练完之后，我们的排长对我们刚才的表现不满意了，起因是有个别人不认真，团长当着我们的面公开批评了他，于是又开始了新一轮的加练。在这整个过程中我们都没碰过水，那一个上午真的是达到了我的毅力极限。我开始感觉呼吸困难（呼吸很费力），吸入的空气也发烫，随时随地都有瘫倒在地的可能。周围也有一些同学开始毫无征兆地倒地，我很害怕自己会想他们一样不省人事，好在我最终还是坚持住了。

白天我们安排的是最简单的军事训练，军姿和踢正步这种，各个学校都差不多，没什么好写的。晚上的团建就令人印象深刻了，大体是做一些游戏，然后输的人进行才艺展示。还有一些同学会上去主动展示，有的表演街舞，还有的吹奏各种乐器。我人生中的第一次音乐会也是在这时体验的：当有人在操场中央唱歌时，周围的同学会主动拿出手机打开闪光灯打call，还有不少会跟着演唱者一起唱。这都是我之前没有见过的光景，我被这种场景深深地打动了，我开始觉得这才是青春，这才是真正的大学生活。

军训中最值得期待的环节就是实弹射击，不过体验并没有我想象中的那么好。开枪的时候，你身后的安全员会死死地按着你，不让你做多余的动作。你唯一能做的就是趴下，扣动扳机。我在射击的时候完全看不出我打倒哪里了，八成是射到草地上脱靶了。最后教官给每人发了一枚弹壳留做纪念。



# 英语分班

高中的时候我最擅长的就是英语，但是到大学里我有点水土不服了。大学的英语考试和高中的英语考试差别很大，比如考试时得自带收音机（用来听听力）。我们英语的分班考试是在军训期间进行的，许多人汗流浃背地就去考场了，而我则是简短地洗了个澡才出发，~~我觉得这是我能进快班的重要原因~~。

我们的分班考试有一个非常迷惑的地方，虽然要求带收音机，但是考场还是为给你广播听力的音频文件。这就导致了用收音机的同学会受到考场外放的听力的干扰。更致命的是，这些听力题只放一遍，和高考完全不同，我在听力部分就开始犯难了。不过你难大家也难，在听第一题的时候，我刻意环视了考场，发现基本上没人动笔，所以我也放平了心态。值得一提的是，我旁边还坐了一个啃苹果的妹子，这个妹子在考试的时候一直在啃苹果，差点把我心态搞乱了。好在我最后还是顺利地通过了分班考试，进入了快班进行学习。



# 业务拓展部

军训期间，有院学生会的干事来寝室上门宣传。抱着体验大学生活的想法，我拉着几个舍友一起报了名。当时我对他们宣传的各个部门完全没有概念，于是随缘选了一个业务拓展部。

在业务拓展部我进行了人生中的第一次面试。面试时学长也没有问什么特别高深的问题，主要是聊聊天，还提了几个问题，比如业务拓展部是做什么，如果你的想法和部长冲突了怎么办。我真不知道业务拓展部是做什么的，所以随便扯了一点。后来zh部长介绍说我们部门主要负责汉姆看包和传信，说实话我还是没搞懂，感觉这个部门就是想办法找点事做。

进了业务拓展部之后我也没干啥活，就参加了几次会议，讨论讨论接下来的工作啥的。印象深刻的活动就只有各种聚餐了，我还记得迎新会上我们玩卧底游戏，院学生会聚餐上各个部长在台上抢凳子。由于我对业拓部实在是没有贡献，就不多浪费笔墨了。转专业后我自动脱离了业拓部，结束了我简短的学生会经历。



# ROBOCON机器人

社团是大学生活非常重要的一部分，刚进入大学的我也想参加社团。然后我就看到了宿舍楼底下贴着的机器人队伍的招新公告，恰好我有过机器人比赛的经验，而且也想参与人工智能的学习（彼时的我认为人工智能就是机器人），所以马上联系了招新的负责人。虽说海报已经贴了，但是学院的新生群里还没有相关的通知，因此可以说我是最早参加机器人队选拔的人之一了。加入迎新群后，勤奋好学的我立刻咨询学长要准备些什么。回复我的是xyk学长，他详细地告诉了我要看《机械制图》的哪几章。

机器人队招新要经过专门的培训和选拔，培训的内容主要分为由机械和控制两大块。因为我有编程基础，所以控制部分上手特别快，甚至还能指出lyl学长的错误。这种温习自己熟悉领域的感觉让我非常舒服，我逐渐有了“把大部分时间都用于编程”的念头。xyk学长看出了我的过人之处，问我要不要直接进队里训练，被我婉拒了，因为我觉得我自己仅仅是有基础，但不一定有天赋。lyl学长则是让我去看看《C Primer Plus》。

虽然我代码写的不错，但是在机械这块我和其他人没什么区别。ROBOCON实际上是一个以机械为主的比赛项目，培训时机械的比重也更大。xyk学长组织了几次组队任务，一开始是3D打印大赛，我们做了一个万里阳光号，我的舍友zyf一个人就完成了所有工作，可能是我太划了，他感受到力不从心，便退出了招新。到后面几次SolidWorks组队时我直接变成了孤家寡人，只好和其他没组的同学凑成一个新的组，当然凝聚力跟其他组肯定是无法相比的。看到其他人都能完成任务，而我们组的人甚至懒得交流，我内心产生了落差感，最终我也选择了退出。

第二次与机器人队邂逅，已经是在大一的暑假里了。那时我刚开始入门人工智能，机器人队也开始了第二次正式的选拔，而且他们传单上写着能带你学习人工智能。抱着学习的心态，我又一次报名了。进队后我才发现，他们的人工智能和大部分学者认为的人工智能有很大差异。机器人的人工智能属于运动智能，而大部分学者眼中的人工智能其实是CV、NLP这些领域。这次招新就没有学长给你讲课了，队里给每一个人发了一大堆STM32的资料让你自学，然后提供免费的板子供你完成各种实验。吐槽一下这些板子的质量实在是太差了，很多bug并不是你代码的问题，而是硬件的原因。我当初就遇到了一个jlink的错误，求助学长，他们表示也无能为力，最后是控制组的领队下场才解决。

我们学校算是冠军的有力争夺者了，因此队伍里的规矩非常严格，每天的作息时间是早八点到晚十点，如果迟到了就得买一枚“原子弹”（就是那种巨型的可乐），有次我接连迟到了好几天，为队里的其他成员贡献了几天免费的饮料；还有就是不能玩手机，领队看到你玩手机就会下来提醒。

这样的日子一直持续了大半个暑假，坚持不下来的人陆陆续续走了，还剩下的就直接进到队伍里面。虽然我坚持到了最后，但是最终没有选择留下。理由有很多：

- 我始终都没能融入队伍。队伍里其他人的交流很频繁，而我很少说话；
- 我和机器人队的理念不一致。我想从事软件方向，机器人队显然给不了我帮助；
- 一些同学在经过一年的培训之后，已经成为了队伍里的骨干，最后也被分到了主控组，而我未能得偿所愿，被分到了定位组（搞陀螺仪和码盘的）。

另外xyk学长的人格魅力也是我继续参加机器人队的重要原因。当初我们班级在参观机器人队的时候，xyk学长当着所有人的面表扬了我，还表达了让我继续参加机器人队的愿望。但是当我过来时他已经去SJTU深造了，那些领队也换成了我不认识的生面孔。我觉得我是在没有什么理由留下了，便选择了退出。现在看来我当初的决定还是挺正确的，在大三那年机器人队如愿拿到了全国冠军，我也大体完成了入学时的目标，成为了我想成为的人。



# 篮球比赛

得益于新高考的制度，我高三有很多空闲时间。我们班的篮球大佬zsj在这段时间一直带着我打篮球，我对这项运动逐渐产生了浓厚的兴趣。升入大学之后，我也带头组织了我们班的篮球群一起约球。适逢新生篮球赛临近，我便把群里的人都拉去参加了。

为了这个篮球赛，我们也是做了一些准备。比赛的前几天晚上我们就一直在商量和实践各种战术（虽然到了场上还是xjb打）。另外还出现了一个小插曲：一开始本来是我打首发，后来其他人在上场前临时决定用身高确定首发，于是我自然沦为了替补。其次我们班没有请教练，球员上场下场都是自己决定的，而我们班同学的体力极其充沛，上了球场就不下来了，所以作为替补的我在整个比赛期间都没什么上场机会，最后随队拿了个第三名。

这是我们队伍里很多人第一次参加篮球比赛，虽然没拿到冠军，但起码我对这个成绩很满意。尝到甜头的我们在大一下学期又参加了院里的篮球赛，这下竞争对手换成大二大三的学长了。小组赛期间我们碰到了去年第一第三的队伍，都被血虐，预赛就没有出线。那个时候我的状态也非常差，不仅赛场上失利，野球场上也是频频碰壁。一天下午我加入了一个高水平的场次，进攻时颗粒无收，防守端被频频点名，我分到哪组哪组就输。后来我打急了，面对对方一次正常的防守，我大喊这是个犯规，防守我的人也没说什么，直接把球给了我。回到寝室后我一直在想这件事，觉得真是不够厚道，从此再也没碰过篮球。



# 回高中宣讲

寒假的时候武大组织了回母校宣讲的活动，参加这种活动还有什么创新学分的奖励。正好我们学院毕业对创新学分有要求，我就找了几个柯中考入武大的同学去做这个活动。但是柯中对外来人员是有很严格的限制的，所以我们的宣讲也没那么顺利，前前后后去了两次。

第一次探望的时候，我们打算找高三的同学宣讲（毕竟高一高二的也没什么宣讲的必要），但是高三的学生课业繁忙，学校不给我们这个机会。经过一番商量后我们决定去高一宣讲交差，不仅是因为高一比较空闲，而且以前高三教我们的老师大都去高一执教了。最终我找到了章琼丰老师，那时她已经是班主任了，正好我们可以去她的班级宣讲。说到这里其实挺惭愧的，其他人探望老师都是带着鲜花和礼物去的，我则是去请老师帮忙的。

那一天我还碰到了fyh，我和他打了会篮球，就顺便去探望其他老师了。首先我们找到了之前的数学老师，当时他在和其他老师打台球。虽然fyh之前还是他的课代表，但是他对俩我们不闻不问，一直在嘻嘻哈哈地打他的台球。我和fyh就在旁边看他打了一局又一局，眼看他又要重开一把，我实在是有点待不住了，打了一声招呼就走了。后面我离校的时候看到他和原来三班的同学在校门口有说有笑，心想果然还是自己的学生亲切一些。其次就是我们当初的班主任，我们在微信群里打了招呼，但是最终并没有等到他。我还单独去探望了物理老师祝建强老师，他也是一改当初严肃的形象，亲切地和我交谈了起来，主要适合我聊大学生活和家庭情况啥的。

几天后学校组织了统一的宣讲，我们又第二次访问了母校。宣讲稿是我晚上在医院赶出来的。当时学校分了三个会场，清华北大一个，浙大一个，其他学校一个，很显然我们就属于「其他学校」。虽然阵势很大，但是来听的人非常少，放眼望去估计都没有20个。后来我觉得宣讲这东西真的没什么必要，很多时候你的分数好学校好专业去不了，差的学校又浪费分数，最后只能去武大，这几乎是唯一的选择，根本不需要我来宣讲，有多少人真的是因为武大的魅力才上的武大？

宣讲期间我还凭借着发传单的借口去其他会场串门了。在那里碰到了几个熟悉的老同学，当然他们有了自己的圈子，我们之间也形同陌生人，下次再遇到他们时我可以表现得更大方一点。



# 转专业

大部分人转计算机是因为计算机待遇好，毕业后月入30k。我对这个数目并没有什么概念，毕竟工作离我还太遥远。我转计算机的理由是它能给我带来虚荣感。打篮球、玩游戏，大家都说我不行，但是谈到代码，大家都认为我很强。因此我觉得我相当适合搞计算机。

由于当年盛行一切专业都转码的风气，所以转计算机专业的竞争非常激烈。当初计算机给了150个名额，但是有300人报名，差不多是2进1的比例。我拍着胸脯对我舍友说，即便是10个人我依然能进。首先我有OI经历，其次我在大学里自学了C、Python和JavaScript编程语言，做了几个爬虫、自动化和游戏项目放在Github上，还看了点MySQL和TCP/IP的书籍，更重要的是我当初的绩点很高，高数还是满分，所以我很有自信。

转专业的时候我被分到了「死亡之组」，据说我们组淘汰率非常高。轮到我的时候已经是最后几个了，主持面试的女老师口气非常严厉，但是我丝毫不慌，毕竟我的履历非常丰富，光是一个自我介绍就耗尽了所有的时间。我一直在滔滔不绝地讲，压根就没给其他老师提问的机会，那个女老师听着听着都笑了。当然时间到了还是被强行打断，一个男老师问我高考差计算机专业几分，我如是回答道1分，这个面试就结束了。几天后我如愿收到了转专业成功的通知。

从我的经历中不难看出，武大转专业还是很轻松的，转计算机只需要一个面试。当然一次简短的面试很难对你的能力有一个全面的评估，这种考核方式是有偶然性的。我们那届就出现了高绩点没过、低绩点过了的现象。

我原先是住在工学部，虽然计算机也属于工科，但是计算机专业的同学是住在单独的信息学部的（然而计算机学院大楼却在文理学部），因此我还要换寝室。信息学部确实有几个空的床位，无奈转过来的人实在是太多了。所以学院就制定了一个方案，分两批换寝室，第一批换的插入13舍的空床位，第二批换的同下一届新生一起入住14舍。13舍和14舍是信部公认最差的两个宿舍楼。其中13舍因为有独卫的存在，风评比14舍略好一些。虽然我对独卫并不感冒，觉得这玩意又脏又占地方，但还是报了第一批换。

换过来住了几天后，我发现信13和之前我住的工5真的没法比。首先就是空间上的区别，工5的寝室非常宽敞，你在床上可以站立，而在信13的床上你只能跪着。其次就是基础设施，你可能很难想象，一个6层楼的宿舍，竟然只有两台洗衣机，如果你要洗衣服就得排很长的队，其中一台坏了情况就更糟（所以我一般都去电信学院的洗衣房洗）。最最重要的，信13总是动不动就停水，我们向宿管阿姨抱怨：怎么又停水了？宿管阿姨答道：14舍也停了。好家伙，搁这比烂呢。所以我在信13待了几天又就回工5了，直到动机的辅导员来赶人了我才回去。

这里介绍下我的新舍友，他们绩点很高，每个人的GPA都在3.9以上，算是一个不折不扣的学霸寝室了。但是我和新舍友的情感是不如我和旧舍友的。我和旧舍友情同手足，重要的日子会一起过，但是新舍友似乎各有各的圈子，每到重要的日子就和别人出去过了，而且他们吃饭时也很少带上我，搬到新的寝室后我就很少说话了。跟这帮学霸相处还是蛮有压力的，这倒不是因为我“不能落后于别人”的思想在作祟，毕竟我对高绩点没有那么强的执念。我的压力是，每次寝室评优的时候，我的GPA都会大幅拉低平均值。后续评奖学金的时候，我还得忍受他们凡尔赛。



# 关于成绩

既然提到绩点了，我们不妨多聊一点。首先谈一谈当时我对绩点的态度：自打进入大学以来，我是从来没有什么“好好学习本科课程，提高绩点，然后保研/出国”之类的想法的。我在高中已经花了相当多的精力去提升成绩了，我不想在大学里重复我高中所做的事情。而且当时一个普遍的认知是：计算机毕业就应该去工作，读研究生就是浪费两/三年。我深以为然。既然我不选择保研/出国，那么绩点也无所谓了，我学我的，你爱给几分就给几分。当然我也没有那种随随便便就满绩的能力，所以我的绩点并不好看。

上文提到我高数考了满分，实际上我大一上学期除了这门外，其他成绩都不是很理想。光89分的课程就有两门，85～89区间的课程更是有5门，占了所有课程的一半还多（总共9门课）。转专业前我认为3.7的绩点已经是很不错了，转过来后我才发现大家都是以满绩为目标，考不到满绩就是失败，这种想法真是太畸形了。

大一下学期的时候，因为我们转专业比较晚，一些专业课他们已经上过几节了，所以还得补课。为了赶进度，原先一周三节的课被补成了一周六节，所以基本上没有空闲的时间了。那段时间我学得很崩溃，好在有ssk和zdt两人陪同。在这种繁忙的节奏中，我逐渐忽视了一些非专业的课程，比如马原。马原是要看慕课的，这个慕课又有时间限制，每隔一段时间就会关闭旧的并开放新的课程。当时我因为忙于其他课程而忘记刷慕课了，恰好授课老师也没提醒。等我记起来才发现1/3的慕课都关了，20分的慕课分我直接丢了6、7分。我们学校GPA采取的是北大算法，分数越低，绩点掉得越快，而马原的平均分又在85上下，你们可以想象我的绩点有多低了。

还有一门课是离散数学。上大学的时候我深知数学的重要性，所以每一门数学课我有都认真对待。为了对付离散数学最后的考试，我还特地去刷了《离散数学与应用》一书的大部分课后题，考试的时候也是得心应手，但是最终没有满绩。刚出成绩的时候，我很诧异，我这么努力才88分？于是便私聊老师，没想到老师并没有直接回答我，而是在群里说对分数有疑问别来找他，然后就把群解散了。

马原和离散，这两门课程让我改变了对绩点的态度。本来对绩点我是不置可否的，不重要的课我不会花太多时间，重要的课我自然会认真学。从那时起我开始厌恶这种考核方式。



当初这篇回忆我是公开放在CSDN上，后来浏览量一晚上就破千了，吓得我赶紧设置为仅作者可见。那时我的文笔还非常活泼，人也非常感性，写了很多现在看来没有必要东西，比如在武大的雪、给舍友庆生，还有赏樱等。为了接近我现在的文风，我做了适量的删减，所以部分读者可能会有一种熟悉的陌生感。不得不承认我确实被计算机专业腌入味了，三年前我测MBTI的时候还是ESTJ，三年后我已经是INTP了。
